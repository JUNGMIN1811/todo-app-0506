# Todo App

바닐라 자바스크립트로 만든 간단한 할 일 관리 앱입니다.

## 실행 방법

별도의 설치나 빌드 없이 `index.html` 파일을 브라우저에서 바로 열면 실행됩니다.

## 주요 기능

- 할 일 추가 — 입력창에 텍스트를 입력 후 **추가** 버튼 또는 `Enter` 키로 등록
- 완료 예정 시간 설정 — 날짜·시간을 선택하면 항목에 함께 표시
- 완료 토글 — 항목 텍스트를 클릭하면 완료/미완료 상태 전환
- 항목 삭제 — `×` 버튼으로 삭제
- 탭 필터 — **전체보기 / 진행중 / 완료** 탭으로 목록 필터링
- 새로고침 후에도 데이터 유지 — `localStorage`에 자동 저장

## 파일 구조

```
todo-app/
├── index.html   # 정적 마크업
├── style.css    # 스타일
└── app.js       # 전체 로직 및 상태 관리
```

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- localStorage
